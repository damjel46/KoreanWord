# 🇰🇷 문해력 퀴즈 (KoreanWord)

> **당신의 어휘력을 테스트하고 향상시키세요!**
> 초성 퀴즈를 통해 어려운 어휘를 재미있게 학습하고, 타임어택 모드로 전 세계 유저들과 경쟁하는 안드로이드 앱입니다.

<br>

## 🛠 사용 기술 스택 (Tech Stack)
| 구분 | 기술 / 라이브러리 |
| :--- | :--- |
| **Language** | Java |
| **Platform** | Android (Min SDK: 24) |
| **UI** | XML Layout, Material Components |

| **Service** | Google Play Games Services v2 (Leaderboard) |
| **Ads** | Google AdMob (Banner Ads) |

<br>

## ✨ 주요 기능 (Key Features)

### 1. 챌린지 모드 (Time Attack)
* **모드 분할:** **1분** 및 **3분** 도전 모드를 구현하고 별도의 최고 점수로 관리.
* **패널티 시스템:** 오답 시 **-2초** / PASS 시 **-5초**의 시간 차감 패널티 적용.
* **리더보드 연동:** **GPGS**를 통한 3분 챌린지 점수 서버 제출 및 랭킹 조회 기능 구현.
* **안전 저장:** 게임 도중 이탈(`onDestroy`) 시에도 신기록 자동 저장.

### 2. 학습 모드 (Study Mode)
* **그룹 학습:** 초성 그룹별(ㄱ-ㅁ, ㅂ-ㅊ, ㅋ-ㅎ) 학습 기능 제공.
* **데이터 관리:** 즐겨찾기(⭐) 및 단어 제외(🗑️) 기능 구현.
* **타이머:** 학습 시간 측정을 위한 스톱워치 기능 상단 배치.
